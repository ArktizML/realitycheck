# Goal Tracker

A productivity web app built with FastAPI that helps users track goals, deadlines and progress.

## Features

- User authentication
- Create and manage goals
- Deadlines with overdue detection
- Hall of Fame for completed goals
- Level system (Initiate → Elite)
- Completion milestones
- Overdue counter in navbar

## Tech Stack

- FastAPI
- SQLAlchemy
- PostgreSQL
- Jinja2
- SQLite
- HTML / CSS
- Progress tracking
- Statistics dashboard
- Goal categories
- Better notifications
